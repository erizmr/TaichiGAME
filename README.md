## Introduction
TaichiGAME(**Taichi Graphics Acceleate Motion Engine**) now is a simple 2d rigid body physics simulation engine powered by [taichi lang](https://github.com/taichi-dev/taichi).

## Requirement
1. python3.7+

    2. support type hints

2. taichi0.7+
